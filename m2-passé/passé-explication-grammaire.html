📚 Les temps du passé de l'Indicatif (passé composé, imparfait, plus que parfait)

Pourquoi plusieurs temps du passé ?

En français, on utilise différents temps pour exprimer le passé selon :

  • la durée de l’action
  • si l’action est terminée ou en cours
  • l’ordre chronologique
Je regardais la télé quand il est arrivé.
➡️ action en cours + action ponctuelle

L’imparfait

Utilisé pour :

  • décrire une situation ou un contexte
  • exprimer une habitude
  • une action en cours dans le passé
Quand j’étais petit, je jouais au foot.
Il faisait beau et les oiseaux chantaient.
💡 Imparfait = description / habitude / durée

Le passé composé

Utilisé pour :

  • une action ponctuelle
  • une action terminée
  • une succession d’actions
J’ai mangé, puis je suis parti.
Il est arrivé à 20h.
💡 Passé composé = action terminée, précise

Le plus-que-parfait

Utilisé pour :

  • une action antérieure à une autre action passée
  • exprimer “le passé du passé”
Quand il est arrivé, j’avais déjà mangé.
Elle a compris qu’il avait menti.
💡 Plus-que-parfait = action avant une autre dans le passé

Comment les former ?

  • Imparfait : radical + terminaisons (-ais, -ait, -ais, -ions, -iez, -aient)
  • Passé composé : auxiliaire (présent) + participe passé
  • Plus-que-parfait : auxiliaire (imparfait) + participe passé
je parlais / j’ai parlé / j’avais parlé

Le choix de l’auxiliaire (être ou avoir)

Pour le passé composé et le plus-que-parfait, on utilise généralement avoir. Mais on utilise être dans deux cas précis :

1. Les verbes pronominaux

Tous les verbes qui commencent par "se" ou "s'".

Je suis allé (Aller) ➡️ Je me suis lavé (Se laver).
Il était parti ➡️ Il s'était souvenu.

2. Les verbes de la "Maison d'Être"

Une liste limitée de verbes de mouvement ou d'état (souvent résumés par l'acronyme DR & MRS VANDERTRAMP) :

  • Devenir, Revenir
  • Monter, Rester, Sortir
  • Venir, Aller, Naître, Descendre, Entrer, Rentrer, Tomber, Retourner, Arriver, Mourir, Partir.
⚠️ Accord : Avec l'auxiliaire être, on accorde toujours le participe passé avec le sujet !
Elle est allée. Ils sont partis.

Imparfait (parler)

je parlais

tu parlais

il parlait

nous parlions

vous parliez

ils parlaient

Plus-que-parfait (aller)

j’étais allé(e)

tu étais allé(e)

il était allé

nous étions allés

vous étiez allé(s)

ils étaient allés

Imparfait vs Passé composé

  • Imparfait : contexte, durée
  • Passé composé : action principale
Je lisais quand il a téléphoné.
➡️ lire = en cours / téléphoner = interruption

Chronologie des actions

On combine souvent les trois temps :

  • plus-que-parfait → action la plus ancienne
  • imparfait → contexte
  • passé composé → action principale
Il pleuvait quand je suis sorti parce que j’avais oublié mon parapluie.

⚠️ Pièges fréquents

  • confondre description et action ❌
  • oublier le plus-que-parfait pour l’antériorité ❗
❌ Quand il est arrivé, je mangeais déjà.
✅ Quand il est arrivé, j’avais déjà mangé.
