⏳ La Concordance des Temps

Qu’est-ce que la concordance des temps ?

C’est la relation entre le temps du verbe principal et le temps du verbe dans la subordonnée.

  • On adapte le temps pour garder une logique dans le temps
  • Très important avec que
Je pense qu’il vient. (présent + présent) ✅
Je pensais qu’il venait. (passé + imparfait) ✅
💡 Le temps dépend du verbe principal
💡 On “recule” souvent le temps dans le passé

Présent → Présent / Futur / Passé composé

  • Présent + présent : action actuelle
  • Présent + futur : action future
  • Présent + passé composé : action terminée
Je dis qu’il travaille. ✅
Je dis qu’il travaillera. ✅
Je dis qu’il a travaillé. ✅

Passé → Imparfait / Conditionnel / Plus-que-parfait

  • Imparfait : action en cours dans le passé
  • Conditionnel : futur dans le passé
  • Plus-que-parfait : action déjà terminée
Je disais qu’il travaillait. ✅
Je disais qu’il travaillerait. ✅
Je disais qu’il avait travaillé. ✅
💡 Futur dans le passé → conditionnel
💡 Action avant le passé → plus-que-parfait

Attention aux changements

  • présent → imparfait
  • futur → conditionnel
  • passé composé → plus-que-parfait
Il dit : “Je partirai” → Il a dit qu’il partirait.
Il dit : “Je finis” → Il a dit qu’il finissait.

Présent → Passé

Je pense qu’il vient

→ Je pensais qu’il venait

Futur → Conditionnel

Je pense qu’il viendra

→ Je pensais qu’il viendrait

⚠️ Pièges fréquents

  • Ne pas garder le futur après un verbe au passé ❌
  • Attention aux traductions littérales de l’anglais
❌ Il a dit qu’il viendra
✅ Il a dit qu’il viendrait
