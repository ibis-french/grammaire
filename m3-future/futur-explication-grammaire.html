📚 Les temps du futur de l'Indicatif (futur simple, futur antérieur)

Pourquoi plusieurs temps du futur ?

En français, on utilise différents temps pour exprimer le futur selon :

  • le déroulement ou la ponctualité de l'action
  • l'ordre chronologique entre les actions
Quand je serai arrivé, je t'appellerai.
➡️ action achevée avant le commencement de la suivante

Le futur simple

Utilisé pour :

  • exprimer une action future
  • faire une prédiction
  • prendre une décision
Demain, il fera beau.
Nous irons au parc cet après-midi.
💡 Futur simple = action future / prédiction

Le futur antérieur

Utilisé pour :

  • une action antérieure à une autre action future
  • exprimer “le passé du futur”
Dès qu'il aura mangé, il partira.
Quand nous aurons fini, nous regarderons un film.
💡 Futur antérieur = action achevée avant une autre dans le futur

Comment les former ?

  • Futur simple : l'infinitif du verbe + terminaisons (-ai, -as, -a, -ons, -ez, -ont)
  • Futur antérieur : l'auxiliaire (*avoir* ou *être*) au futur simple + participe passé
je parlerai / j'aurai parlé / je serai parti(e)

Le choix de l'auxiliaire (être ou avoir)

Pour le futur antérieur, on utilise généralement avoir. Mais on utilise être dans les deux mêmes cas qu'au passé composé :

1. Les verbes pronominaux

Verbes qui commencent par "se" ou "s'".

Je me serai lavé.
Ils se seront souvenus.

2. Les verbes de la "Maison d'Être"

Une liste limitée de verbes de mouvement ou d'état (résumés par l'acronyme DR & MRS VANDERTRAMP) :

  • Devenir, Revenir
  • Monter, Rester, Sortir
  • Venir, Aller, Naître, Descendre, Entrer, Rentrer, Tomber, Retourner, Arriver, Mourir, Partir.
⚠️ Accord : Avec l'auxiliaire être, on accorde toujours le participe passé avec le sujet !
Elle sera partie. Ils seront revenus.

Futur simple (parler)

je parlerai

tu parleras

il parlera

nous parlerons

vous parlerez

ils parleront

Futur antérieur (aller)

je serai allé(e)

tu seras allé(e)

il sera allé

nous serons allés

vous serez allé(s)

ils seront allés

Futur simple vs Futur antérieur

  • Futur antérieur : action accomplie en premier
  • Futur simple : action qui suit
Dès qu'il aura fini ses devoirs, il sortira avec ses amis.
➡️ finir les devoirs (futur antérieur) ➡️ sortir (futur simple)

⚠️ Pièges fréquents

  • oublier le futur antérieur pour l'antériorité ❗
  • utiliser le futur simple après "quand" (en français, l'action est exprimée au futur antérieur).
❌ Quand je finirai, je partirai.
✅ Quand j'aurai fini, je partirai.
